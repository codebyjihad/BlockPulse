# Blog App Frontend

A modern frontend for a blog application built with **React**, **Vite**, **TailwindCSS**, and **React Router**. Users can view, create, edit, and delete blog posts. Forms are handled with **React Hook Form** for easy validation and management.

---

## Features

- View all blog posts  
- Create new blog posts  
- Edit existing blog posts  
- Delete blog posts  
- Responsive UI with TailwindCSS  
- Form handling with React Hook Form  
- Navigation with React Router  

---

## Tech Stack

- **Frontend:** React, TypeScript, Vite  
- **Styling:** TailwindCSS  
- **Routing:** React Router  
- **Forms:** React Hook Form  
- **Icons:** React Icons  
- **Linting:** ESLint
